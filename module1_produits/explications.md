# Module 1 – Gestion des Produits

## Objectif
Mettre en place un système CRUD (Create, Read, Update, Delete) pour gérer les produits des commerçants.

## Fonctionnalités
- Ajouter un produit (id, nom, description, prix, quantité)
- Lire la liste des produits depuis un fichier CSV
- Modifier un produit existant
- Supprimer un produit
- Gérer les erreurs (fichier manquant, données invalides)

## Bonnes pratiques
- Toujours valider les données avant insertion
- Utiliser des tests unitaires pour vérifier les cas limites
- Ne jamais stocker de données sensibles en clair
